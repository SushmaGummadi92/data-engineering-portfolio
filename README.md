## Project 1: Customer Data ETL Pipeline
**Tools:** Python, pandas  
**Dataset:** Kaggle Customer Personality Analysis (2,240 records)  
**File:** practice.py

### Pipeline steps:
1. Extract — load raw CSV data into pandas DataFrame
2. Profile — analyse shape, nulls, distributions
3. Clean — impute 24 missing Income values using mean
4. Transform — map education categories via lookup join
5. Export — save cleaned and filtered datasets to CSV

### Key results:
- 24 null Income values detected and fixed
- 1,156 high-income customers identified
- 1,293 no-kids customers identified  
- 5 education categories standardized
